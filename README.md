# Binary to Gray Code Converter

A **Binary to Gray Code Converter** is a combinational logic circuit that transforms a standard binary number into its corresponding Gray code equivalent. 

Gray code is a non-weighted, unit-distance code where only one bit changes at a time between successive values. This property is essential in digital systems to minimize switching errors and glitches during state transitions.

---

## 1. Conversion Logic
To convert an $n$-bit binary number ($B_{n-1} \dots B_0$) to a Gray code ($G_{n-1} \dots G_0$), the following rules apply:

1.  **MSB Preservation:** The Most Significant Bit of the Gray code is identical to the MSB of the binary number[cite: 1, 2].
    *   $G_{n-1} = B_{n-1}$
2.  **XOR Operation:** Each subsequent Gray code bit is generated by performing a Bitwise XOR operation between the current binary bit and the bit immediately to its left[cite: 1, 2].
    *   $G_i = B_{i+1} \oplus B_i$

---

## 2. 4-Bit Truth Table
The following table demonstrates the conversion for 4-bit values:

| Binary ($B_3 B_2 B_1 B_0$) | Gray Code ($G_3 G_2 G_1 G_0$) |
| :---: | :---: |
| 0000 | 0000 |
| 0001 | 0001 |
| 0010 | 0011 |
| 0011 | 0010 |
| 0100 | 0110 |
| 0101 | 0111 |
| 1000 | 1100 |
| 1111 | 1000 |

---

## 3. Logical Expressions
For a 4-bit converter, the Boolean expressions are:

*   **$G_3 = B_3$**[cite: 1, 2]
*   **$G_2 = B_3 \oplus B_2$**[cite: 1, 2]
*   **$G_1 = B_2 \oplus B_1$**[cite: 1, 2]
*   **$G_0 = B_1 \oplus B_0$**[cite: 1, 2]

---

## 4. Implementation Details
The hardware implementation typically uses **(n-1) XOR gates** for an $n$-bit converter. In a 4-bit design, three XOR gates are used to process the bits while the MSB is passed through directly.



---

## 5. Applications
*   **Rotary Encoders:** Used to prevent erroneous readings in mechanical position sensing[cite: 1, 2].
*   **Asynchronous FIFOs:** Essential for safe pointer crossing between different clock domains[cite: 1, 2].
*   **Error Minimization:** Reduces noise and power consumption by limiting bit-switching in high-speed digital buses[cite: 1, 2].
